# 🖥️ Desktop Environments

Set `DE=` in `tx11start` and `debian-setup.sh` to your choice.

---

## XFCE (Recommended)
```bash
DE="xfce"
```
- Lightweight, stable, best for gaming
- Whiskermenu start menu included
- Stock theme: Greybird

## KDE Plasma
```bash
DE="kde"
```
- Feature-rich, beautiful
- ⚠️ Heavy — needs 6GB+ RAM

## GNOME
```bash
DE="gnome"
```
- Modern UI
- ⚠️ Heavy — needs 6GB+ RAM

## MATE
```bash
DE="mate"
```
- Classic desktop feel
- Medium RAM usage

## LXQt
```bash
DE="lxqt"
```
- Very lightweight Qt-based DE
- Good for low-end devices

## Openbox
```bash
DE="openbox"
```
- Ultra minimal window manager
- Right-click desktop menu

## i3wm
```bash
DE="i3wm"
```
- Tiling window manager
- Keyboard-driven workflow

## Cinnamon
```bash
DE="cinnamon"
```
- Windows-like layout
- Medium RAM usage

---
